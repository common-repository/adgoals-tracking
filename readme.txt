AdGoals Tracking lets you connect WooCommerce with AdGoals.

== Description ==

AdGoals Tracking helps you easily connect WooCommerce with AdGoals to track affiliate sales.

= Docs & Support =

You can get support and assistance by contacting [AdGoals Support](https://adgoals.net/how-can-we-help-you/).

= Required Plugins =

The following plugins are required to use AdGoals Tracking:

* [WooCommerce](https://wordpress.org/plugins/woocommerce/) by Automattic

== Installation ==

1. Upload the entire `adgoals-tracking` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'AdGoals Tracking' menu under 'WooCommerce' in your WordPress admin menu.
